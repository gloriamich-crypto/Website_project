#  Learn Floristry: A Journey into Kenyan Blooms

A responsive, visually engaging web project designed to showcase different flower varieties and arrangements found in Kenya. This site serves as an educational resource for aspiring florists or anyone interested in the local floral industry.

##  Features

* **Interactive Navigation:** A clean navbar with a centered logo and a hover-activated dropdown menu for categorized flower types.
* **Dynamic Grid Layout:** A custom-built 3-up, 2-down flower card display using **CSS Flexbox**.
* **Responsive Design:** Optimized for different screen sizes, ensuring the "Feel Free to Contact Us" header and flower cards remain perfectly centered.
* **Kenya-Focused Content:** Highlighting popular flowers like Roses, Gladioli, and Carnations common in the Kenyan market.
* **Interactive Styling:** Pink-themed borders and subtle hover effects that make the flower boxes "lift" and glow.

## Technologies Used

* **HTML5:** Structured layouts using semantic elements like `<header>`, `<nav>`, and `<section>`.
* **CSS3:** Advanced Flexbox techniques for centering, wrapping, and spacing elements.
* **Git:** Version control used for tracking changes and managing code updates.
* **GitHub:** Remote hosting for the repository.

## How to Run Locally

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/gloriamich-crypto/Website_project.git](https://github.com/gloriamich-crypto/Website_project.git)
    ```
2.  **Navigate to the folder:**
    ```bash
    cd Website_project
    ```
3.  **Launch the site:**
    Open `web2.html` or `contacts.html` in your favorite web browser.

 ##  Project Structure

* `web2.html`: Main landing page with the flower arrangement grid.
* `contacts.html`: A clean contact page for user inquiries.
* `web2.css`: The primary stylesheet containing the Flexbox logic and pink-themed UI.
* `README.md`: Project documentation.
